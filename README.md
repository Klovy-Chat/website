
### 3. Install Dependencies

Install the project dependencies using your preferred package manager:

Using **pnpm**:
```bash
pnpm install
```

```bash
npm install
```

Using **yarn**:
```bash
yarn install
```

### 4. Start the Development Server
Using **pnpm**:
```bash

```
Using **npm**:
```bash
npm run dev
```

```bash

```
